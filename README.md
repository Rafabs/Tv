# 📺 Lab+ TV - Programação Interativa

Este projeto é uma aplicação web desenvolvida para a exibição dinâmica da grade de programação da **Lab+ TV**. Ele foi criado exclusivamente para fins de **estudo, testes de integração e aprimoramento técnico** em arquiteturas modernas de front-end.

---

## 🚀 Sobre o Projeto

O objetivo principal deste repositório é migrar uma estrutura legível de HTML/JS puro para o framework **Astro**, explorando benefícios como:
* **Componentização:** Divisão da interface em peças reutilizáveis.
* **Zero JS por padrão:** Otimização de performance entregando o mínimo de JavaScript para o cliente.
* **TypeScript:** Implementação de tipagem estática para evitar erros em tempo de execução.
* **Manipulação de Dados:** Consumo de dados dinâmicos via arquivos JSON.

## 🛠️ Tecnologias Utilizadas

* [Astro](https://astro.build/) - Framework web focado em performance.
* [TypeScript](https://www.typescriptlang.org/) - Superset de JavaScript para tipagem segura.
* [CSS3](https://developer.mozilla.org/pt-BR/docs/Web/CSS) - Estilização moderna com Glassmorphism e animações neon.
* [JSON](https://www.json.org/json-pt.html) - Estrutura de dados para a grade de programação.

## 📁 Estrutura de Pastas Chave

* `src/pages/`: Contém as rotas e o layout principal da aplicação.
* `src/scripts/`: Lógica de manipulação do DOM e filtros de data (TypeScript).
* `src/data/`: Centralização das informações de programação em arquivos JSON.
* `src/styles/`: Arquivos de estilo globais e específicos.
